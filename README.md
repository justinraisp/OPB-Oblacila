# OPB-Oblacila
Naredili bomo spletno bazo za modne artikle. Baza bo vključevala artikle in njihove lastnosti kot so:
* Barva,
* Stil,
* Velikost,
* Količina,
* Velikostni tip,
* Proizvajalec,
* Status,
* Ali je artikel na zalogi.

Spletna aplikacija bo omogočola vnašanje artiklov.
